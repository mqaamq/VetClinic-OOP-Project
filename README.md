# VetClinic-OOP-Project
===Veterinary Clinic Management System===

Project Description:
This project is a simple Veterinary Clinic Management System developed in Java as part of the Object-Oriented Programming (OOP) course.
The system demonstrates the use of core OOP principles such as:
- Encapsulation
- Classes and Objects
- Constructors
- Getters and Setters
- Method interaction between classes

Project Structure:
The project consists of the following classes:
Pet – stores information about pets and provides methods to analyze their age and life stage.
Owner – represents pet owners and tracks the number of pets they have.
Veterinarian – stores veterinarian details and determines treatment eligibility based on specialization and experience.
Main – the entry point of the program where objects are created and tested.

Technologies Used:
- Java
- IntelliJ IDEA
- Git & GitHub

How to Run:
- Clone the repository:
  git clone https://github.com/your-username/vetclinic_project.git
- Open the project in IntelliJ IDEA
- Run the Main.java file

Features Implemented:
- Create and manage pets, owners, and veterinarians
- Determine pet life stage and age category
- Check if a veterinarian can treat a specific pet
- Identify frequent clinic clients
- Demonstrate getters, setters, and additional methods
